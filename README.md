#Digital Clock Dark/Light

#Project Description
A real-time digital clock built with HTML, CSS, and JavaScript that displays hours, minutes, and seconds. 
It supports both light and dark modes, which can be toggled with a single click. The clock also shows AM/PM.

#Features
- Real-time updating of hours, minutes, and seconds
- 12-hour format with AM/PM indicator
- Toggle between dark and light themes
- Responsive design, works on all screen sizes
- Stylish UI with shadow effects and smooth transitions

#Technologies Used
- HTML5
- CSS3 (with Google Fonts & Font Awesome)
- JavaScript (Vanilla JS)

#Future Improvements
- Add 24-hour clock option
- Add alarm functionality
- Add customizable themes/colors
- Add seconds animation
